# Vectr Scroll Experience

A scroll-driven Three.js and GSAP landing page inspired by the visual motion
language of Vectr's website. The experience combines a procedural 3D scene,
camera movement, staged content transitions, and a glowing path synchronized
to page scroll.

## Live Demo

[vectr-scroll-experience.netlify.app](https://vectr-scroll-experience.netlify.app)

## Built With

- React
- Three.js
- GSAP and ScrollTrigger
- Vite

## Local Development

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

## Deployment

The repository includes `netlify.toml` and can be deployed directly to
Netlify. The production output is generated in `dist/`.
